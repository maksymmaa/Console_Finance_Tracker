# Console Finance Tracker

A lightweight, interactive command-line Python application designed to help track personal expenses, categorize purchases, and strictly enforce spending limits. 

## Features
* **Interactive Menu:** A user-friendly console loop to navigate features without restarting the program.
* **Log Purchases:** Record items with their specific name, price, and custom category.
* **Spending Limits:** Set an overall financial limit. The program will actively block new purchases if they exceed the current balance.
* **Filter by Category:** Easily isolate and view specific types of expenses (e.g., "Food", "Transport").
* **Calculate Balance:** Instantly view a breakdown of all purchase prices and the total sum spent.

## Prerequisites
* **Python 3.10 or higher** is required (this project utilizes structural pattern matching `match/case`).
